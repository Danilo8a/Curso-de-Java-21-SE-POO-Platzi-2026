# Repositorio del curso de Java 21 SE de Platzi (2026)
Este es el proyecto que se desarrolla durante el curso, llamado Platzi Live, una app para alojar contenido multimedia como películas y documentales.
## Herramientas aprendidas durante el curso
Durante la realización del curso, se estudiaron los conceptos básicos del lenguaje, tales como su funcionamiento, la JVM y la POO. Entre los aspectos destacados esta:

- Programación orientada a objetos: abstracción, encapsulamento, herencia y polimorfismo
- Casting de tipos y clases envolventes de primitivos
- Variables y constantes
- Uso de la clase List y de una de sus implementaciones: ArrayList
- Estructuras de control: if-else, while y switch
- Streams y lambdas
- Uso de enums
- Excepciones checked y unchecked
- Mapas en Java
- Lectura y escritura de archivos planosClases y métodos abstractos
- Creación e implementación de interfaces
- Records para clases preestablecidas
